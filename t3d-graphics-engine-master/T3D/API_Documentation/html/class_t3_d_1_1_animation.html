Detailed Description

Animation component. Ticks every frame until completion, updating a rigid skeletal hierarchy of Bones associated with an GameObject.

This class is just one way to implement an Animation. An alternative is to subclass Task, which allows for a more centralized way to manage multiple objects, or even multiple Animations.

Positions in between each timestep are linearly interpolated, and orientations are spherically interpolated.
