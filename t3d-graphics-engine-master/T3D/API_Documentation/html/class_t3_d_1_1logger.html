static

Initialise the logging subsystem. Must be called before using any other functions.

Parameters
PriorityAll priorities equal or greater to this will be shown. As in, Info implies Warning and Error.
OutputWhich output stream/s should the logger initialize.
EnableAllCategoriesShould all application categories be filtered initially? (e.g. Platform, Audio, Animation, etc)
LogFilePathWhere the log file is found. Assumed to be relative to where the T3D executable is running from. No support for UTF8/UTF16 paths.
